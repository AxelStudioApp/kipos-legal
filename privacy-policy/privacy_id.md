# Kebijakan Privasi kiPOS
**Terakhir Diperbarui: Maret 2026**

Kebijakan Privasi ini menjelaskan bagaimana **Axel Studio** ("kami") mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan layanan dan aplikasi kiPOS. Kami berkomitmen penuh untuk menjaga kerahasiaan data bisnis Anda.

---

### 1. Informasi yang Kami Kumpulkan
Kami mengumpulkan beberapa jenis informasi untuk memberikan layanan POS yang optimal:
* **Data Akun Bisnis:** Nama bisnis, alamat, nomor telepon, alamat email, dan pengaturan pajak/layanan yang Anda berikan saat registrasi.
* **Data Operasional:** Informasi produk, resep (BOM), data karyawan (Shift/PIN), dan catatan transaksi penjualan yang Anda proses melalui aplikasi.
* **Data Perangkat dan Telemetri:** Alamat IP, jenis sistem operasi, model perangkat, serta status keamanan perangkat (termasuk deteksi *Root*/*Jailbreak*) untuk mencegah penipuan dan memastikan integritas transaksi.

### 2. Bagaimana Kami Menggunakan Informasi Anda
Data yang dikumpulkan akan digunakan secara eksklusif untuk:
* Mengoperasikan, memelihara, dan menyediakan fungsionalitas aplikasi kiPOS.
* Memproses sinkronisasi data *offline* ke *cloud* secara aman.
* Memberikan dukungan pelanggan (*Customer Support*) dan mengirimkan pemberitahuan terkait pemeliharaan server atau pembaruan wajib (*Forced Update*).
* Menganalisis metrik penggunaan secara anonim untuk meningkatkan performa UI/UX aplikasi.

### 3. Berbagi dan Pengungkapan Data
**Kami tidak menjual data bisnis atau transaksi Anda kepada pihak ketiga.** Data Anda hanya dapat dibagikan dalam kondisi berikut:
* **Penyedia Layanan (Infrastruktur):** Kami menggunakan layanan *cloud hosting* pihak ketiga yang memiliki sertifikasi keamanan standar industri untuk menyimpan data Anda.
* **Kepatuhan Hukum:** Kami dapat mengungkapkan data Anda jika diwajibkan oleh hukum, perintah pengadilan, atau lembaga pemerintah yang berwenang.

### 4. Keamanan Data
Kami menerapkan standar keamanan teknis yang ketat, termasuk enkripsi jalur komunikasi (HTTPS/SSL) dan perlindungan basis data (*database*). Namun, tidak ada transmisi data melalui internet yang 100% aman. Kami tidak dapat memberikan jaminan mutlak atas keamanan data dari serangan siber tingkat lanjut.

### 5. Retensi dan Penghapusan Data
Kami menyimpan data operasional Anda selama akun Anda aktif. Jika Anda memutuskan untuk berhenti berlangganan, Anda dapat meminta penghapusan seluruh data bisnis dari server kami dengan menghubungi tim dukungan Axel Studio.
